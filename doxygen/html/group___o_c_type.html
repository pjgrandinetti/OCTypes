Typedef Documentation

◆ OCBase

typedef struct __OCBase OCBase

The fundamental base structure for all OCTypes.

This structure is embedded as the first member in all OCType-compatible objects. It holds common information like the type identifier, retain count, and function pointers for essential operations.
