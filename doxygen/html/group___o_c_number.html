typedef const struct __OCNumber* OCNumberRef

.. _OCNumber_h:

OCNumber.h

Defines the OCNumberRef type and related functions for representing various numerical types.

OCNumber provides a way to work with different numerical data types (integers, floats, complex numbers) as OCType objects. It allows for creation and inspection of these numerical objects. :type OCNumberRef: :brief: An opaque reference to an immutable OCNumber object.

OCNumberRef is used to represent various numerical values within the OCTypes framework. These objects encapsulate primitive number types and provide a common interface for them.
