Detailed Description

.. _OCString_h:

OCString.h

Immutable and mutable OCString reference types and operations.

Provides creation, manipulation, comparison, and conversion functions for OCTypes string objects, including support for complex arithmetic parsing.

:def STR: :brief: Macro to create a constant OCStringRef from a C string literal. STR("...") is implemented as a compile-time constant OCString that you do not own, so you must not call OCRelease on it. It lives for the lifetime of your process.
