Start comparison from the end of the string.

Comparison is anchored to the starting point.

kOCCompareNonliteral 

Perform loose equivalence (e.g., o-umlaut == o, umlaut).

kOCCompareLocalized 

Use the user's default locale for comparisons.

kOCCompareNumerically 

Perform numeric comparison (e.g., Foo2.txt < Foo7.txt < Foo25.txt).

kOCCompareDiacriticInsensitive 

Ignore diacritics during comparison (e.g., o-umlaut == o).

kOCCompareWidthInsensitive 

Ignore width differences (e.g., 'a' == UFF41).

kOCCompareForcedOrdering 

Force an ordering if strings are equivalent but not strictly equal (for sorting stability).
